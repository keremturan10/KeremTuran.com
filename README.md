# KeremTuran.com - Kişisel Blog

Bu repository Kerem Turan'ın kişisel blog web sitesini içermektedir.

## 📋 İçerik

- **index.html** - Ana sayfa (blog yazılarının listelenmesi)
- **about.html** - Hakkımda sayfası (kişisel bilgiler, yetenekler)
- **contact.html** - İletişim formu (ziyaretçilerle iletişim)
- **style.css** - Tasarım ve stil dosyası
- **script.js** - JavaScript işlevleri

## 🎨 Özellikler

✅ Responsive tasarım (mobil uyumlu)
✅ Modern ve temiz kullanıcı arayüzü
✅ İletişim formu
✅ Blog yazı kartları
✅ Sosyal medya linkleri
✅ Smooth scrolling
✅ Form validasyonu

## 🚀 Başlangıç

1. Repository'yi klonlayın:
```bash
git clone https://github.com/keremturan10/KeremTuran.com.git
```

2. Dosyaları bir web sunucusunda açın veya `index.html`'i tarayıcıda açın.

## 📁 Dosya Yapısı

```
KeremTuran.com/
├── index.html
├── about.html
├── contact.html
├── style.css
├── script.js
└── README.md
```

## 🎯 Kullanılan Teknolojiler

- HTML5
- CSS3 (Responsive Design, Grid, Flexbox)
- JavaScript (ES6+)

## 📝 Özelleştirme

### E-mail Adresi Değiştirme
`contact.html` dosyasında aşağıdaki satırı bulun:
```html
<a href="mailto:kerem@example.com">kerem@example.com</a>
```

Kendi e-mail adresinizle değiştirin.

### Sosyal Medya Linkleri
Tüm sayfaların altında yer alan sosyal medya linklerini düzenleyin:
```html
<a href="#">Twitter</a>
<a href="#">GitHub</a>
<a href="#">LinkedIn</a>
```
